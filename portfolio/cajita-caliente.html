Cajita Caliente

Full Stack JavaScript Project

Goals

The objectives of the project were to:

  • Work in a simulated team project environment, managing team contributions through Github.
  • Craft thoughtful userstories together, as a team.
  • Implement CRUD functionality in a RESTful Node.js API.
  • Store application data in a NoSQL database.
  • Consume the API via AJAX in a front-end built with jQuery.
  • Layout and style your application with well-formatted CSS
  • Allow users to login via third-party accounts using oAuth.

Project Idea

Cajita Caliente is an app that allows users to upload files into a virtual file system. Ordinary users can only read/download a file where as Owners can do anything to the files they own. In addition to keeping track of the file structure, this app also associates meta-data with each file, including items like: date when the file was either created or uploaded, date it was modified, and tags.

Tools and Results

The technologies used for this project on the frontend, the App uses Javascript, JQuery, HTML, CSS, Bootstrap, Gitflow, and Flexbox. To help stay on top of tasks and responsibilities, the team used Trello Board.

Regarding the backend include such technologies as: Node modules, Express, and Mongo DB.
