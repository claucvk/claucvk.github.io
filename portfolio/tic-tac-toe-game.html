Tic Tac Toe Game

Full Stack Project

Goals

The objectives of the project were to create an interactive game in the browser and apply foundational programming skills to game logic.

Tools and Results

I builded the front-end of the game using technologies like HTML, CSS, SCSS, Bootstrap, AJAX, jQuery, Javascript. In addition I created some users stories to understand the game requirements as well as a wireframe to define the UI with two different screens: Initial screen and Screen after sign in.
