Mkchromecast

UX Design (UX) / User Interface Design (UI)

Goals

This is a program to cast your MacOS audio or Linux audio to Google Cast devices. Mkchomecast's goal was to create an interface that would be easy and simple to use. The idea was to offer default settings addressed to novice users, but being flexible enough to allow expert users to configure the program with other advanced features.

Tools

We applied some UX methods to accomplish this objective:

  1. We established the mkchromecast's goals and the expectations of its founder.
  2. We defined who were the users, their goals, needs, and context. Then, among potential users, we selected two groups of target users.
  3. We developed use cases to understand the interaction between the user and the mkchromecast (the user's tasks and the corresponding program responses).
  4. We defined the information architecture of the program. We chose the vocabulary and the organization of content that would fit with the mental model of users.
  5. We released a beta version of mkchromecast to get the user's feedback. Since then, users have reported issues that have been taking in account to improve the performance of this program.

Results

Thanks to the combination of a fine UX and programming work, currently Mkchromecast has more than 10,000 happy users (Mac OS and Linux). Even, Uk's omg ubuntu recommend it.
