Interactive Greeting Card

CCS Animation

Goals

The objective was to create an interactive greeting card using HTML and CSS. It was a final project for a web programming course that I was doing.

Tools

  1. I looked for some ideas on internet to inspire my design.
  2. I wrote a pseudo code to define the card features and the user interaction.
  3. I used HTML to define the structure of the website and to build some images.
  4. I used CSS to define the style and animations of the card as well as to position the parts of the images that I built.
  5. Finally, I defined the visual design of the card (colors, topography and element positioning).

Results

The result was a greeting card which opens when the users hover over it to show a dog's message. Inside, first, you find the instruction on how to read the message. Then you find an animation that makes the dog jump. Finally, at the bottom, there are two text boxes with a typing effect to simulate a translation from the dog's language to English.
